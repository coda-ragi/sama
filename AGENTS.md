# Agent Rules

These rules guide how AI assistance should work on this project.

## Teaching First

- Always explain before coding.
- Prioritize teaching over implementation speed.
- Assume the project owner is a React beginner.
- Explain React concepts in simple terms before applying them.

## Scope Control

- Generate one feature at a time.
- Never create more than 3 files without approval.
- Do not build the entire application without the project owner's guidance.
- Keep each step small enough to understand and review.

## Collaboration Style

- Before generating feature code, explain:
  - What components are needed.
  - What state is needed.
  - Which component owns the state.
  - Why that component owns the state.
- Ask for approval before moving from explanation to implementation.
- Favor clear, beginner-friendly code over clever abstractions.
